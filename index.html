Your Domain Name

A domain name is essentially the web equivalent of a postal address. If a person wishes to find out more about a company by looking on a web browser (Internet Explorer, Google Chrome, Firefox etc.), they need to know the address of the company's website. This "address is the domain name" e.g. the domain name for Microsoft company is microsoft.com. Click here to learn more.
